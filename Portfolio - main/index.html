Trvel Treasury

Built using HTML, CSS, JavaScript, Node.js, and MongoDB, it features a dynamic table that updates in real-time with each expense entry and stores all data persistently in a MongoDB database.

Memory Game

The Memory Game is a web-based application using HTML, CSS, and JavaScript. Players flip cards to find matching emoji pairs, with visual feedback for matches and non-matches. The game includes a shuffle function to randomize cards and a reset button for restarting the game.
